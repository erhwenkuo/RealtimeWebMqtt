Build RealTime WebApp

This is a demonstration web project which will use below software stack to build a realtime web app:

Front End

  1. MVVM framework - angular.js
  2. Style Layout - bootstrap.js
  3. PubSub Messaging - mqttws31.js

Back End

  1. WebServer - Node.js
  2. Storage - Redis
  3. Messaging - MQTT Broker (Mosquitto)
